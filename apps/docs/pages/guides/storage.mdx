OpenModels Storage makes it simple to upload and serve files of any size, providing a robust framework for file access controls.

## Features

You can use OpenModels Storage to store images, videos, documents, and any other file type. Serve your assets with a global CDN to reduce latency from over 285 cities globally. OpenModels Storage includes a built-in image optimizer, so you can resize and compress your media files on the fly.
