OpenModels generates documentation in the [Dashboard](https://open-models-platform.com/dashboard) which updates as you make database changes.

1. Go to the [API](https://open-models-platform.com/dashboard/project/_/api) page in the Dashboard.
2. Select any table under **Tables and Views** in the sidebar.
3. Switch between the JavaScript and the cURL docs using the tabs.
